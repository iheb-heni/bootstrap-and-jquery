Tutoriel : Typographie avec Bootstrap

1. Taille du texte

Bootstrap offre plusieurs classes pour ajuster la taille du texte.

Exemples de tailles de texte :

Texte très grand (fs-1)

Texte assez grand (fs-2)

Texte moyen (fs-3)

Texte plus petit (fs-4)

Texte encore plus petit (fs-5)

Texte très petit (fs-6)

<p class="fs-1">Texte très grand (fs-1)</p>

2. Poids du texte

Vous pouvez changer le poids de la police pour la rendre plus légère ou plus épaisse.

Exemples de poids de texte :

Texte léger (fw-light)

Texte normal (fw-normal)

Texte gras (fw-bold)

<p class="fw-light">Texte léger (fw-light)</p>

3. Alignement du texte

Bootstrap permet de facilement aligner le texte à gauche, à droite, ou au centre.

Exemples d'alignement du texte :

Texte aligné à gauche (text-start)

Texte centré (text-center)

Texte aligné à droite (text-end)

<p class="text-start">Texte aligné à gauche (text-start)</p>

4. Décoration du texte

Vous pouvez ajouter des décorations comme des soulignements, des barrés ou des italiques.

Exemples de décoration du texte :

Texte souligné (text-decoration-underline)

Texte barré (text-decoration-line-through)

Texte en italique (fst-italic)

<p class="text-decoration-underline">Texte souligné (text-decoration-underline)</p>

5. Famille de police

Bootstrap vous permet de définir différentes familles de polices.

Exemples de familles de police :

Texte avec police sans-serif (font-sans-serif)

Texte avec police serif (font-serif)

Texte avec police monospace (font-monospace)

<p class="font-sans-serif">Texte avec police sans-serif (font-sans-serif)</p>

Tutoriel Bootstrap Icons

1. Introduction aux Bootstrap Icons

Les Bootstrap Icons sont une bibliothèque d'icônes open-source de haute qualité qui peuvent être utilisées facilement dans vos projets. Vous pouvez les intégrer à votre projet en ajoutant le CDN de Bootstrap Icons ou en les installant via npm.

Pour inclure Bootstrap Icons, ajoutez simplement la ligne suivante dans la balise <head> de votre HTML :

<link href="https://cdn.jsdelivr.net/npm/bootstrap-icons/font/bootstrap-icons.css" rel="stylesheet">

2. Affichage des icônes

Pour afficher une icône, utilisez la balise <i> avec le nom de classe correspondant à l'icône que vous souhaitez. Voici un exemple :

<i class="bi bi-alarm"></i>

3. Icônes avec texte

Vous pouvez également combiner des icônes avec du texte pour un meilleur visuel. Utilisez la balise <i> aux côtés de votre texte dans n'importe quel élément HTML.

<i class="bi bi-heart"></i> J'aime Bootstrap

Résultat :

J'aime Bootstrap

4. Changer la taille des icônes

Vous pouvez ajuster la taille des icônes en utilisant les classes fs-*. Voici quelques exemples :

  • fs-1 - Taille la plus grande
  • fs-2 - Taille grande
  • fs-3 - Taille moyenne
  • fs-4 - Taille petite
<i class="bi bi-heart fs-1"></i> Icône Grande

Résultat :

Icône Grande

5. Couleurs des icônes

Les icônes Bootstrap héritent de la couleur de leur élément parent. Vous pouvez changer la couleur de l'icône en utilisant les classes utilitaires text-*. Voici comment faire :

  • text-primary - Couleur bleue
  • text-secondary - Couleur grise
  • text-success - Couleur verte
  • text-danger - Couleur rouge
<i class="bi bi-heart text-danger"></i> Icône Rouge

Résultat :

Icône Rouge

6. Rotation des icônes

Vous pouvez faire pivoter les icônes en utilisant les classes rotate-*. Voici comment :

  • rotate-90 - Rotation de 90 degrés
  • rotate-180 - Rotation de 180 degrés
  • rotate-270 - Rotation de 270 degrés
<i class="bi bi-arrow-clockwise rotate-90"></i> Icône Rotée

Résultat :

Icône Rotée
