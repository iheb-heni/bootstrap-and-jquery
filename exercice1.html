Cette formation couvre les bases de Bootstrap et jQuery, deux outils essentiels pour les développeurs front-end. Vous apprendrez à créer des interfaces modernes et dynamiques.

Bootstrap : Exemple d'une Grille

Voici un exemple de mise en page avec la grille Bootstrap :

                    
<div class="container">
    <div class="row">
        <div class="col-md-6">Colonne 1</div>
        <div class="col-md-6">Colonne 2</div>
    </div>
</div>

jQuery : Sélecteurs

Utilisez les sélecteurs jQuery pour cibler des éléments HTML :

                    
$(document).ready(function() {

  $("button").click(function() {

    $("#message").text("Bonjour, jQuery !");

  });

});
