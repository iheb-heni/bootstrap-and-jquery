Tutoriel : Inline et Bloc avec Bootstrap

Tableau des éléments Inline et Bloc

Voici un tableau pour mieux comprendre la différence entre les éléments Inline et Bloc en HTML :

Type d'Élément Description Exemples
Inline
  • Ne prend que la largeur de son contenu
  • Peut être placé côte à côte avec d'autres éléments
<span>, <a>, <strong>, <img>
Bloc
  • Occupe toute la largeur de son conteneur
  • Commence sur une nouvelle ligne
 <div>, <p>, <h1> à <h6>, <ul>

1. Comprendre les éléments Inline

Les éléments inline ne prennent que la largeur de leur contenu. Avec Bootstrap, vous pouvez ajouter des styles comme .d-inline pour les afficher côte à côte.

Exemple :
Inline 1 Inline 2
<span class="d-inline">...</span>

2. Comprendre les éléments Bloc

Les éléments bloc occupent toute la largeur de leur conteneur. Bootstrap offre .d-block pour forcer ce comportement.

Exemple :
Bloc 1 Bloc 2
<span class="d-block">...</span>
