Comprendre le Système de Grille de Bootstrap

1. Introduction au système de grille

Le système de grille de Bootstrap est basé sur 12 colonnes. Cela signifie que l’espace d’une ligne est divisé en 12 parties égales. Vous pouvez utiliser ces colonnes pour ajuster la disposition de votre contenu. La structure de base de la grille est composée de deux éléments principaux :

  • .container : Conteneur pour maintenir une largeur maximale et de la mise en page.
  • .row : Contient les colonnes et détermine leur disposition horizontale.

2. Exemple de Grille de Base (12 Colonnes)

Voici une grille simple où chaque colonne occupe 4 espaces (3 colonnes total, chaque colonne occupe 4/12).

4 Colonnes
4 Colonnes
4 Colonnes
<div class="col-4">...</div> (répété trois fois)

3. Colonnes Responsives

Bootstrap vous permet de rendre vos colonnes responsives, c’est-à-dire qu'elles vont changer de taille en fonction de la largeur de l’écran. Les classes suivantes sont utilisées pour cela :

  • .col-* : Pour les petits écrans (moins de 576px).
  • .col-sm-* : Pour les écrans de petite taille (à partir de 576px).
  • .col-md-* : Pour les écrans de taille moyenne (à partir de 768px).
  • .col-lg-* : Pour les grands écrans (à partir de 992px).
  • .col-xl-* : Pour les écrans très larges (à partir de 1200px).

Exemple de grille réactive (les colonnes changent selon la taille de l’écran) :

Colonne 1
Colonne 2
Colonne 3
<div class="col-12 col-md-6 col-lg-4">...</div>
<div class="col-12 col-md-6 col-lg-4">...</div>
<div class="col-12 col-md-12 col-lg-4">...</div>

4. Espacement Entre les Colonnes

Bootstrap permet de créer un espacement entre les colonnes à l’aide des classes de marge sur la ligne. Par exemple, .g-3 ajoute un espacement de 1 rem entre les colonnes.

4 Colonnes
4 Colonnes
4 Colonnes
<div class="row g-3">...</div>

5. Décalages des Colonnes

Un décalage est un espace vide ajouté avant une colonne, ce qui permet de décaler son contenu sans ajouter de marges externes. Exemple : un décalage de 4 colonnes avec la classe .offset-4.

Colonne Décalée de 4
<div class="col-4 offset-4">...</div>

6. Colonnes Imbriquées

Vous pouvez imbriquer des colonnes à l’intérieur d’une autre colonne pour créer des mises en page plus complexes. Cela permet de créer des dispositions avec des sous-grilles.

Colonne Principale
Colonne Imbriquée 1
Colonne Imbriquée 2
Colonne Secondaire
<div class="col-8">...</div>
<div class="col-6">...</div>
<div class="col-6">...</div>

7. Résumé des Classes de Grille

  • .col-* : Définit une colonne occupante de l’espace restant dans la ligne.
  • .col-sm-* : Pour les petits écrans (à partir de 576px).
  • .col-md-* : Pour les écrans moyens (à partir de 768px).
  • .col-lg-* : Pour les grands écrans (à partir de 992px).
  • .col-xl-* : Pour les écrans très larges (à partir de 1200px).
  • .offset-* : Décale une colonne d’un nombre donné de colonnes.
  • .g-* : Ajoute un espacement global entre les colonnes.
