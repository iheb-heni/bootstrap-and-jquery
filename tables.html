Bootstrap Tables Tutorial

1. Basic Table

To create a simple table, use the .table class.

# Name Age City
1 John Doe 25 New York
2 Jane Smith 30 Chicago
<table class="table">
  <thead>...</thead>
  <tbody>...</tbody>
</table>

2. Striped Rows

Add the .table-striped class for striped rows.

# Name Age City
1 John Doe 25 New York
2 Jane Smith 30 Chicago
<table class="table table-striped">...</table>

3. Bordered Table

Add the .table-bordered class to add borders to all cells.

# Name Age City
1 John Doe 25 New York
2 Jane Smith 30 Chicago
<table class="table table-bordered">...</table>

4. Hoverable Rows

Add the .table-hover class to highlight rows on hover.

# Name Age City
1 John Doe 25 New York
2 Jane Smith 30 Chicago
<table class="table table-hover">...</table>

5. Small Table

Add the .table-sm class to make a more compact table.

# Name Age City
1 John Doe 25 New York
2 Jane Smith 30 Chicago
<table class="table table-sm">...</table>

6. Responsive Table

Wrap the table with .table-responsive to make it scrollable on smaller devices.

# Name Age City Country Occupation Department Contact
1 John Doe 25 New York USA Developer IT 123-456-7890
2 Jane Smith 30 Chicago USA Designer Creative 987-654-3210
<div class="table-responsive"><table>...</table></div>

7. Dark Table

Use .table-dark for a table with a dark background.

# Name Age City
1 John Doe 25 New York
2 Jane Smith 30 Chicago
<table class="table table-dark">...</table>
